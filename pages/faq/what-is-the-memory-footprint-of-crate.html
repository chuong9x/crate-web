title: What is the memory footprint of Crate? link: https://crate.io/faq/what-is-the-memory-footprint-of-crate.html author: Nils Magnus description: Answer about frequently asked question about requirements, configuration. created: 2016-05-17 post_name: what-is-the-memory-footprint-of-crate status: publish post_type: faq tags: requirements weight: 1 The memory footprint of Crate depends mostly on the results (and size) of a query. Crate doesn't have a traditional index structure retained in memory, but instead uses the inbuilt caching that Lucene leverages from the Operating System. We have a more [detailed guide to memory handling here](https://crate.io/docs/reference/configuration.html#crate-heap-size).
