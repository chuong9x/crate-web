title: Can I use Kibana with Crate? link: https://crate.io/faq/can-i-use-kibana-with-crate.html author: Nils Magnus description: Answer about frequently asked question about elasticsearch, kibana, visualization. created: 2016-05-17 post_name: can-i-use-kibana-with-crate status: publish post_type: faq weight: 1 Yes, we have users that [connect to Crate with Kibana](http://techblog.bigdata-analyst.de/2015/06/25/how-to-use-kibana-4-with-crate-io/) with the `es.api`. We are currently adding support for Metabase, which is a query manager and visualization BI tool. There are other visualization tools also available.
