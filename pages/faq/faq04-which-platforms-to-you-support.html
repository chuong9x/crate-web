title: Which platforms to you support? link: https://crate.io/faq/faq04-which-platforms-to-you-support.html author: Nils Magnus description: Answer about frequently asked question about requirements, configuration. created: 2016-05-17 post_name: faq04-which-platforms-to-you-support status: publish post_type: faq weight: 1 Crate runs wherever a Java runtime is available. A support contract with SLAs is available for a number of Enterprise Linux distributions, we call Core Platforms. For base platforms Crate is supported on a best effort base.
