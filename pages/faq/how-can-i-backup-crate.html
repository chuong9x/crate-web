title: How can I backup Crate? link: https://crate.io/faq/how-can-i-backup-crate author: Chris Ward description: How can I backup Crate? created: 2016-05-17 post_name: how-can-i-backup-crate status: publish post_type: faq tags: backup weight: 2 Crate has a convenient `CREATE SNAPSHOT` SQL command that stores the current state of one or all tables into a backup store. There are drivers for several stores available, including local file system, an Amazon S3 storage, or HDFS. For more details see the [documentation](/docs/reference/sql/snapshot_restore.html).
