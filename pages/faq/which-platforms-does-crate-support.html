title: Which platforms does Crate support? link: https://crate.io/faq/which-platforms-does-crate-support.html author: Nils Magnus description: Answer about frequently asked question about requirements, configuration. created: 2016-05-17 post_name: which-platforms-does-crate-support status: publish post_type: faq tags: requirements weight: 1 The Crate core is written in Java and runs whenever a Java Runtime Environment (up from Java 8) is available. Additional to the base package, there are self contained packages available for several Linux distributions, container, virtualization and cloud environments.
