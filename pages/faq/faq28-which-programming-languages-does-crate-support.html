title: Which programming languages does Crate support? link: https://crate.io/faq/faq28-which-programming-languages-does-crate-support.html author: Nils Magnus description: Answer about frequently asked question about language. created: 2016-05-17 post_name: faq28-which-programming-languages-does-crate-support status: publish post_type: faq weight: 1 There are driverse for many programming languages including Java (JDBC), Python, and PHP are supported and are freely available. The ODBC driver is available with an Enterprise License. More drivers for Ruby, Erlang, Scala, Node.js, Go, and Perl are community contributions.
