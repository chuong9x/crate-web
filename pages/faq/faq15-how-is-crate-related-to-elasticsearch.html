title: How is Crate related to Elasticsearch? link: https://crate.io/faq/faq15-how-is-crate-related-to-elasticsearch.html author: Nils Magnus description: Answer about frequently asked question about elasticsearch. created: 2016-05-17 post_name: faq15-how-is-crate-related-to-elasticsearch status: publish post_type: faq weight: 3 Crate is a database, which Elasticsearch is a search engine. Crate uses some infrastructure from Elasticsearch and a few other open source components, but adds substantial functionality on top of that.
