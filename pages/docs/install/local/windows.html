title: Install Crate on Windows author: Chris Ward ## Configure Windows To install Crate on Windows, you will need a [Java JDK installed](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html). Set the `JAVA_HOME` environment variable: ![Windows environment variables]({% media '/media/1512/win-envvar.png' %}) Allow the ports Crate uses through the Windows Firewall: ![Windows Firewall ports]({% media '/media/1512/win-port.gif' %}) ## Install Crate [Download the Crate Tarball](/docs/install/local/tarball/), expand it and move to a convenient location. Start Crate by running `.\bin\crate`, to create a cluster, run the command multiple times. ## Next Steps - [Connect to Crate Cluster](/docs/connect) - [Import Data](/docs/import) - [Client Libraries](/docs/clients) - [Scale a Cluster](/docs/scale)
